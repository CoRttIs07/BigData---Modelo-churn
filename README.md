# BigData - Modelo Churn

Proyecto de Big Data enfocado en la predicción de retiro de asociados
de una cooperativa financiera mediante técnicas de Machine Learning.

## Objetivo

Predecir la probabilidad de que un asociado se retire en los próximos 6 meses,
permitiendo generar estrategias de retención temprana.

## Arquitectura

- Bronze: datos crudos
- Silver: transformación y construcción de variables
- Gold: modelo predictivo y visualización

## Tecnologías

- Python
- PySpark
- Pandas
- Scikit-learn
- VS Code
- GitHub
